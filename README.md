# NeoUserStats
WIP Custom User Stats Generator

## Running
To run the script run: 
- `ruby .\main.rb`  
- `ruby .\main.rb <username>`  

To run a debug test run:
- `ruby .\main.rb -c`

## Requirements
- faraday: `gem install faraday-retry`  
- octokit: `gem install octokit`
